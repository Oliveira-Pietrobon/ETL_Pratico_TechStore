<h2>📊 Aula Teste — Integração de Dados e ETL</h2>
<h4>Projeto prático desenvolvido em Python para demonstrar um pipeline de ETL (Extract, Transform, Load) utilizando múltiplas fontes de dados.</h4>

<h4>O notebook foi criado como uma atividade guiada, com etapas progressivas para aprendizado de:</h4>

<ol>
  <li>Manipulação de dados com Pandas</li>
<li>Integração entre arquivos CSV, Excel e banco SQLite</li>
<li>Limpeza e transformação de dados</li>
<li>Geração de análises e relatórios</li>
<li>Automação básica de pipelines de dados</li>

<h2>🚀 Tecnologias Utilizadas</h2>
<ul>
  <li>Python 3</li>
  <li>Pandas</li>
  <li>SQLite</li>
  <li>SQLAlchemy</li>
  <li>OpenPyXL</li>
  <li>Matplotlib</li>
  <li>Jupyter Notebook</li>
</ul>

<h2>🔧 Funcionalidades</h2>
<h3>O projeto realiza:</h3>
<ul>
  <li>✅ Criação automática de dados fictícios</li>
  <li>✅ Leitura de múltiplas fontes de dados</li>
  <li>✅ Integração entre tabelas e arquivos</li>
  <li>✅ Transformações e tratamento de dados</li>
  <li>✅ Criação de métricas e análises</li>
  <li>✅ Exportação de resultados</li>
  <li>✅ Visualização de dados com gráficos</li>
</ul>

<h2>🏗️ Pipeline ETL</h2>
<h3>1️⃣ Extract (Extração)</h3>
<h3>Os dados são extraídos de:</h3>
<ul>
  <li>Banco SQLite (clientes)</li>
  <li>Arquivo CSV (pedidos.csv)</li>
  <li>Arquivo Excel (produtos.xlsx)</li>
</ul>
<h3>2️⃣ Transform (Transformação)</h3>
<ul>
  <li>Limpeza de dados</li>
  <li>Conversão de tipos</li>
  <li>Junção de tabelas</li>
  <li>Tratamento de valores ausentes</li>
  <li>Criação de métricas</li>
  <li>Organização dos dados</li>
</ul>
<h3>3️⃣ Load (Carga)</h3>
<h3>Os dados processados são:</h3>
<ul>
  <li>Salvos novamente no banco</li>
  <li>Exportados para arquivos</li>
  <li>Utilizados em análises e visualizações</li>
</ul>
